Данная программа решает задачи линейного программирования в предмете математическое моделирование симплекс методом, где искомая функция стремится к max. Я писал этот код по алгоритму моего преподавателя по мат моделированию, вашей задачи он может не подойти или просто не решить ее без видимой на это причины, которую я не смог предусмотреть. Не используйте этот код как что-то важное. Если найдете баг пишите мне в соц. сети, покумекаем что не так вместе.

Кратко объясню что вводить и что получаешь.

Вводим мы матрицу, пример:

	x1	x2	x3	x4	x5	b
a1	1	2	1	0	0	14
a2	-5	3	0	1	0	15
a3	4	6	0	0	-1	24
z	1	1	0	0	0	


Мы получили ее из системы неравенств, где искомая функция z:
 -
/ x1 + 2x2 <= 14
< -5x1 + 3x2 <= 15
\ 4x1 + 6x2 >=24
 -
z=x1+x2 z=>max

Вводим в программу:

4
6
2
1 2 1 0 0 14
-5 3 0 1 0 15
4 6 0 0 -1 24
1 1 0 0 0

Первая строчка это кол-во строк в матрице, вторая строчка число столбцов в матрице, третья строчка это число изначальных переменных, которые мы не "добавляли"
